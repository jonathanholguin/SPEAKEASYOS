Scheduler

This program is a sheduler simulator that implements the First Come, First Serve (FCFS), Shortest Job First (SJF), and Shortest Remaining Time First (SRTF) scheduling policies.

*NOTICE*
The following simulator was coded using Visual Studio (Not Visual Studio Code).
To open the program, clone the respository or download the latest file.
Open the project folder in Visual studio.
Run the program.

*HOW TO USE*

The user will be prompted to enter a number of processes, Process ID, Arrival Time, and Burst time.
The program will then display the Completion time, turnaround time, and waiting time.
